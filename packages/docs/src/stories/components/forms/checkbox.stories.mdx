# Checkbox

> Checkbox component used for selecting multiple values from several options.
> And you can easily customize and extends the Box component with `css` prop or `asChild` prop
>
> #### Unique Props :
>
> - `boxSize`, change fontSize and checkbox size
> - `colorScheme`, change color of the checkbox, includes background checked and border when hovered

## Example Usage

```jsx
import { Checkbox } from '@zero-ui/core';
```
