# SideMenu

Used to create sidebar navigation. Usually used for dashboar page or admin page.

## Basic Usage

To use the `<SideMenu />` component you just need to follow the code as below :

```tsx
<SideMenu>
  <SideMenu.Item title="Menu 1" />
  <SideMenu.Item title="Menu 2" />
  <SideMenu.Item title="Menu 3" />
</SideMenu>
```

## With Icon

You can build the menu with icon by using the `icon` prop. You can use any icon from `@ant-design/icons` or any other icon library. The icon will be placed on the left side of the menu title.

```tsx
<SideMenu>
  <SideMenu.Item title="Menu 1" icon={<SettingOutlined />} />
  <SideMenu.Item title="Menu 2" />
  <SideMenu.Item title="Menu 3" />
</SideMenu>
```

## With Sub Menu

You can also build the menu with sub menu by using the `<SideMenu.Sub />` component like the following code :

```tsx
<SideMenu>
  <SideMenu.Item title="Menu 1" icon={<SettingOutlined />} />
  <SideMenu.Item title="Menu 2" />
  <SideMenu.Item title="Menu 3" />
  <SideMenu.Sub title="Menu 4">
    <SideMenu.SubContent>
      <SideMenu.Item title="Menu 4.1" />
      <SideMenu.Item title="Menu 4.2" />
      <SideMenu.Item title="Menu 4.3" />
    </SideMenu.SubContent>
  </SideMenu.Sub>
</SideMenu>
```

## API

### `<SideMenu />`

| Name  | Type      | Default | Description            |
| ----- | --------- | ------- | ---------------------- |
| title | string    |         | Set the SideMenu title |
| icon  | ReactNode |         | Set the SideMenu icon  |

### `<SideMenu.Item />`

| Name  | Type      | Default | Description                 |
| ----- | --------- | ------- | --------------------------- |
| title | string    |         | Set the SideMenu.Item title |
| icon  | ReactNode |         | Set the SideMenu.Item icon  |

### `<SideMenu.Sub />`

| Name  | Type      | Default | Description                        |
| ----- | --------- | ------- | ---------------------------------- |
| title | string    |         | Set the SideMenu.Sub title         |
| icon  | ReactNode |         | Set the SideMenu.Sub icon          |
| value | string    |         | Required props for radix accordion |
