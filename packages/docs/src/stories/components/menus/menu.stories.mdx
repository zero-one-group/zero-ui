# Menu

This component can be used to create navigation, dropdown menus and so on.

## Basic Usage

To use the `<Menu />` component you just need to follow the code as below :

```tsx
<Menu title="Menu">
  <Menu.Item title="Menu Item 1" />
  <Menu.Item title="Menu Item 2" />
  <Menu.Item title="Menu Item 3" />
</Menu>
```

The code above will result in a display like this :

<Menu title="Menu">
  <Menu.Item title="Menu Item 1" />
  <Menu.Item title="Menu Item 2" />
  <Menu.Item title="Menu Item 3" />
</Menu>

## With icon

To use the `<Menu />` component with icon you just need to follow the code as below :

```tsx
import { SettingOutlined } from '@ant-design/icons';

export default function MenuDisplay() {
  return (
    <Menu title="Menu With Icon" icon={<SettingOutlined />}>
      <Menu.Item title="Menu With Icon Item 1" />
      <Menu.Item title="Menu With Icon Item 2" />
      <Menu.Item title="Menu With Icon Item 3" />
    </Menu>
  );
}
```

The code above will result in a display like this :

<Menu title="Menu With Icon" icon={<SettingOutlined />}>
  <Menu.Item title="Menu With Icon Item 1" />
  <Menu.Item title="Menu With Icon Item 2" />
  <Menu.Item title="Menu With Icon Item 3" />
</Menu>

## With Sub Menu

To use the `<Menu />` component with sub menu you just need to follow the code as below :

```tsx
<Menu title="Menu">
  <Menu.Item title="Menu Item 1" />
  <Menu.Item title="Menu Item 2" />
  <Menu.Item title="Menu Item 3" />
  <Menu title="Menu Item 4">
    <Menu.Item title="Menu Item 4.1" />
    <Menu.Item title="Menu Item 4.2" />
    <Menu.Item title="Menu Item 4.3" />
  </Menu>
</Menu>
```

The code above will result in a display like this :

<Menu title="Menu">
  <Menu.Item title="Menu Item 1" />
  <Menu.Item title="Menu Item 2" />
  <Menu.Item title="Menu Item 3" />
  <Menu title="Menu Item 4">
    <Menu.Item title="Menu Item 4.1" />
    <Menu.Item title="Menu Item 4.2" />
    <Menu.Item title="Menu Item 4.3" />
  </Menu>
</Menu>

You can also use icon in sub menu with the following code :

```tsx
import { SettingOutlined } from '@ant-design/icons';

export default function MenuDisplay() {
  return (
    <Menu title="Menu">
      <Menu.Item title="Menu Item 1" />
      <Menu.Item title="Menu Item 2" />
      <Menu.Item title="Menu Item 3" />
      <Menu title="Menu Item 4" icon={<SettingOutlined />}>
        <Menu.Item title="Menu Item 4.1" />
        <Menu.Item title="Menu Item 4.2" />
        <Menu.Item title="Menu Item 4.3" />
      </Menu>
    </Menu>
  );
}
```

The code above will result in a display like this :

<Menu title="Menu">
  <Menu.Item title="Menu Item 1" />
  <Menu.Item title="Menu Item 2" />
  <Menu.Item title="Menu Item 3" />
  <Menu title="Menu Item 4" icon={<SettingOutlined />}>
    <Menu.Item title="Menu Item 4.1" />
    <Menu.Item title="Menu Item 4.2" />
    <Menu.Item title="Menu Item 4.3" />
  </Menu>
</Menu>

## API

### `<Menu />`

| Name      | Type                                                                                                                                                       | Default                         | Description        |
| --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------- | ------------------ |
| title     | string                                                                                                                                                     | undefined                       | Set the Menu title |
| icon      | ReactNode                                                                                                                                                  | undefined                       | Set the Menu icon  |
| placement | bottom \| left \| right \| top \| bottom-end \| bottom-start \| left-end \| left-start \| right-end \| right-start \| top-end \| top-start \| bottom-start | Set the menu dropdown placement |

### `<Menu.Item />`

| Name  | Type      | Default   | Description             |
| ----- | --------- | --------- | ----------------------- |
| title | string    | undefined | Set the Menu.Item title |
| icon  | ReactNode | undefined | Set the Menu.Item icon  |
