# Breadcrumb

Displays the current location within a hierarchy.

# Structures

Breadcrumbs uses a fully composable API to freely customize each component.

## `<Breadcrumb />`

The `Breadcrumb` root component

| Prop        | Type                | Default | Description                               |
| ----------- | ------------------- | ------- | ----------------------------------------- |
| `separator` | `React.ReactNode`   | `/`     | Set the separator of our breadcrumb items |
| `css`       | `Stitches.CSS`      | -       | Customize breadcrumb style using stitches |
| `children`  | `React.ReactNode[]` | -       | Pass in `<BreadcrumbItem />` components   |

## `<BreadcrumbItem />`

Used to render the breadcrumbs items

| Prop       | Type              | Default | Description                               |
| ---------- | ----------------- | ------- | ----------------------------------------- |
| `css`      | `Stitches.CSS`    | -       | Customize breadcrumb style using stitches |
| `children` | `React.ReactNode` | -       | Render the breadcrumb contents            |

## `<BreadcrumbSeparator />`

The separator can be customized by setting the each separator

> When using BreadcrumbSeparator, the `separator` props must not be passed or set it to `separator=""`.
> Otherwise, the default separator will be rendered

| Prop       | Type              | Default | Description                               |
| ---------- | ----------------- | ------- | ----------------------------------------- |
| `css`      | `Stitches.CSS`    | -       | Customize breadcrumb style using stitches |
| `children` | `React.ReactNode` | -       | Render the breadcrumb separator           |

## Example Usage

```jsx
import { Breadcrumb } from '@zog-ui/core';

// Basic Example
const App = () => (
  <Breadcrumb separator="/">
    <BreadcrumbItem>
      <a href="#">zero-one-group</a>
    </BreadcrumbItem>
    <BreadcrumbItem>
      <a href="#">zero-one-software</a>
    </BreadcrumbItem>
    <BreadcrumbItem>zog-design-system</BreadcrumbItem>
  </Breadcrumb>
);
```
